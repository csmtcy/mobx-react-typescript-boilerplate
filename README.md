  

# React-TypeScript

  

Minimal boilerplate for a single-page app using MobX, React and TypeScript with TSX.

  

Initial run:

  

* Install Node.js

* `npm install`  
* `npm start`
* `npm run dev`
* `npm run watch`
* `npm run build`
